# 毛玻璃风格 (Glassmorphism) 现代化重构方案

根据您的需求，我将为现有博客系统进行一次彻底的视觉与功能重构，核心目标是打造一个具有高度沉浸感、现代感和可定制性的**毛玻璃风格 (Glassmorphism)** 界面。

## 1. 核心视觉设计：Glassmorphism v2.0

我们将采用升级版的毛玻璃风格，不仅仅是简单的模糊，而是结合了**光效、深度、质感**的综合设计语言。

*   **视觉特征**：
    *   **深度模糊**：`backdrop-filter: blur(16px)`，营造磨砂玻璃质感。
    *   **半透背景**：`rgba(255, 255, 255, 0.1)` (亮色) / `rgba(0, 0, 0, 0.2)` (暗色)，保持通透感。
    *   **精致边框**：`1px solid rgba(255, 255, 255, 0.2)`，模拟玻璃边缘的反光。
    *   **噪点纹理**：在玻璃层叠加细微的 Noise 纹理，增加真实质感。
    *   **环境光**：背景中的流体渐变球体 (Gradient Orbs) 在玻璃下方缓慢移动，通过折射产生唯美光影。

## 2. 功能模块：动态背景系统

我们将构建一个强大的背景管理系统，让用户可以自定义整个站点的氛围。

*   **BackgroundManager 组件**：
    *   **状态管理**：使用 Vue 响应式状态管理当前背景类型（Image/Video/Gradient/Color）和源文件。
    *   **预设库**：内置几款高质量的动态渐变和风景视频作为默认选项。
    *   **用户上传**：支持拖拽上传图片或视频，使用 IndexedDB 或 LocalStorage (Base64) 本地存储用户偏好。
    *   **遮罩控制**：提供滑块调节背景遮罩的不透明度 (Overlay Opacity)，确保文字在任何背景下都清晰可读。

## 3. 布局重构：响应式网格与卡片

*   **App.vue 重构**：
    *   将背景层 (`BackgroundManager`) 提升至最顶层，作为 fixed 元素覆盖全屏。
    *   内容层 (`router-view`) 位于背景层之上，采用 `z-index` 分层。
*   **组件玻璃化**：
    *   **Navbar**: 顶部导航栏变为悬浮的玻璃条，滚动时增加模糊度。
    *   **PostCard**: 文章卡片采用玻璃材质，Hover 时上浮并增加光泽感 (CSS `box-shadow` & `border-color` 变化)。
    *   **Sidebar/Modal**: 侧边栏和模态框使用更深度的模糊和遮罩，聚焦用户注意力。

## 4. 技术实现路径

1.  **创建 `BackgroundManager.vue`**：
    *   实现背景渲染逻辑（图片/视频/CSS渐变）。
    *   实现设置面板（玻璃拟态风格的悬浮面板），允许用户切换背景、上传文件、调节遮罩。
    *   持久化存储配置。

2.  **封装 `GlassCard.vue` 组件**：
    *   创建一个通用的玻璃卡片组件，封装重复的 CSS 样式（backdrop-filter, border, shadow）。
    *   支持 `variant` 属性（如 `heavy`、`light`、`interactive`）来控制模糊程度和交互效果。

3.  **全局样式升级 (Tailwind 配置)**：
    *   在 `tailwind.config.js` 中扩展颜色和阴影配置。
    *   定义 `.glass`、`.glass-hover` 等 Utility 类。

4.  **重构现有页面**：
    *   **Home.vue**: 将文章列表改为玻璃卡片网格布局。
    *   **App.vue**: 集成 `BackgroundManager`，调整整体布局结构。
    *   **Post.vue**: 文章阅读页采用“纸张在玻璃上”的设计隐喻，正文区域保持较高的不透明度以保证阅读体验。

## 5. 验证计划

*   **视觉验证**：检查毛玻璃效果在深色/浅色背景下的表现，确保无脏边、文字清晰。
*   **性能验证**：测试背景视频播放时的 CPU/GPU 占用，确保滚动流畅（必要时对模糊区域使用 `will-change: backdrop-filter`）。
*   **兼容性**：检查 Firefox 和 Safari 对 `backdrop-filter` 的支持情况（必要时添加 fallback 背景色）。
