# Markdown 语法提示增强计划

用户希望语法提示不仅仅是静态文本，而是包含"丰富的内容"和"样例演示"。我们将把侧边栏升级为一个交互式的**语法速查手册**。

## 1. 内容结构增强
将现有的简单文本列表升级为结构化数据：
- **标题**: 语法的名称（如"二级标题"）。
- **说明**: 简短的用法说明。
- **源码**: Markdown 原始代码。
- **预览**: 渲染后的 HTML 效果（直接在侧边栏中渲染，所见即所得）。

## 2. UI 设计升级
- **卡片式布局**: 每个语法点作为一个独立的卡片。
- **对比展示**: 上方显示源码（Monospace字体，带背景色），下方显示渲染效果。
- **一键插入**: 点击源码区域，自动将该段代码插入到编辑器光标处（提升实用性）。

## 3. 内容扩充
除了基础语法，增加进阶用法：
- **表格**: 展示 Markdown 表格语法。
- **任务列表**: `- [x] 完成任务`。
- **脚注**: `[^1]`。
- **HTML 标签**: `<small>`, `<kbd>`, `<mark>` 等。
- **数学公式**: (如果支持的话，暂不添加以免增加依赖，保持核心功能纯净)。

## 4. 执行步骤
1.  **定义数据结构**: 在 `Editor.vue` 中定义一个包含所有语法示例的数组对象。
2.  **重构侧边栏 UI**: 使用 `v-for` 渲染语法卡片，实现源码与预览的对照。
3.  **实现插入功能**: 编写 `insertText` 函数，支持点击代码块插入到编辑器。
4.  **样式美化**: 使用 Tailwind 优化卡片阴影、圆角和排版，使其与整体 "Zen Minimalist" 风格保持一致。
