# 重构代码演练场布局与功能

我将对 `CodePlayground.vue` 和 `Post.vue` 进行重构，以满足您提出的新布局和交互需求。

## 1. 核心布局重构 (CodePlayground.vue)
- **取消模态框**：将 `CodePlayground.vue` 从弹窗组件改为内嵌组件，直接嵌入到页面结构中。
- **三栏布局**：整体采用 Flex 布局，包含：
  - **左栏 (文章)**：可折叠侧边栏，宽度可拖拽调整。
  - **中栏 (编辑器)**：代码编辑区，位于右侧主区域的下半部分。
  - **右栏 (预览)**：实时预览区，位于右侧主区域的上半部分。
- **拖拽调整 (Resizable)**：
  - 实现一个水平拖拽条，用于调整左侧文章栏的宽度。
  - 实现一个垂直拖拽条，用于调整右侧预览区和编辑区的高度比例。
- **布局控制**：
  - 添加“展开/收起文章”按钮。
  - 添加“重置布局”按钮，一键恢复默认比例。

## 2. 页面结构调整 (Post.vue)
- **移除悬浮按钮**：删除右下角的“打开代码演练场”悬浮按钮。
- **集成演练场**：
  - 在文章目录上方添加一个“开启代码演练场”按钮。
  - 当点击该按钮时，页面状态切换为“演练模式”。
  - 在演练模式下，原有的文章布局隐藏，显示新的 `CodePlayground` 组件，并将文章内容作为 Slot 或 Props 传入左侧栏。

## 3. 技术实现细节
- **原生拖拽**：不引入额外库，使用 Vue 3 组合式 API + 原生 DOM 事件实现拖拽逻辑，保证轻量高效。
- **状态管理**：使用 `ref` 管理面板宽度/高度百分比（如 `articleWidth` = 25%, `editorHeight` = 50%）。
- **组件通信**：`Post.vue` 将文章内容（HTML）传递给 `CodePlayground`，确保左侧能显示完整的文章阅读体验。

## 4. 验证
- 确认点击目录上方的按钮能正确进入演练模式。
- 确认左侧文章栏可以折叠、展开和拖拽调整宽度。
- 确认右侧预览区和编辑区可以上下拖拽调整高度。
- 确认重置按钮能恢复默认布局。
