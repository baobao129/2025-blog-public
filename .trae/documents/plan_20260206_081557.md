# 为代码编辑器添加 Emmet 和代码片段支持

为了提升代码编辑体验，我将为 `MonacoEditor` 集成 Emmet 插件，并配置常用的代码片段 (Snippets)。

## 1. 安装 Emmet 依赖
- **操作**: 安装 `emmet-monaco-es` 库。
- **原因**: 这是一个轻量级且兼容性好的库，专为 Monaco Editor 提供 Emmet 展开功能（如输入 `div.container>ul>li*3` 后按 Tab 展开）。

## 2. 修改 MonacoEditor 组件 (src/components/MonacoEditor.vue)
- **引入 Emmet**: 在组件挂载时，根据当前语言（HTML/CSS/JSX）注册对应的 Emmet 支持。
- **注册代码片段**:
  - 利用 Monaco Editor 的 `monaco.languages.registerCompletionItemProvider` API。
  - 针对不同语言（HTML, CSS, JS, Vue）预设一些常用的代码片段（Snippets）。
  - 例如：输入 `log` 自动展开为 `console.log($1)`，输入 `vue3` 展开为 Vue 3 的 setup 模板。

## 3. 验证
- **测试 Emmet**: 在 HTML 编辑器中输入缩写（如 `div.card`）并按 Tab，验证是否正确展开。
- **测试 Snippets**: 在 JS 编辑器中输入预设的前缀（如 `clg`），验证是否出现智能提示并能正确补全。
