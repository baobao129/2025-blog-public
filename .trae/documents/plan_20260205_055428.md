# 前端设计升级与功能增强计划

我们将为博客进行全面的视觉升级，采用 **"极简主义 + 杂志排版" (Minimalist Editorial)** 的设计风格，强调排版、留白和细腻的微交互。同时，完善现有的 GitHub 同步功能，确保写作体验流畅。

## 1. 视觉风格定义 (Aesthetic Direction)
- **风格**: 极简主义 (Minimalism) 融合 杂志排版 (Editorial)。
- **字体**: 使用系统字体栈优化后的 Serif (标题) + Sans-serif (正文) 组合，营造阅读质感。
- **配色**: 以暖白/米色为底，深灰/黑为字，搭配一种低饱和度的强调色（如陶土红或鼠尾草绿）。
- **动效**: 页面切换时的淡入淡出，列表项的交错上浮，按钮的细微缩放。
- **布局**: 宽大的边距，非对称的网格，大字体标题。

## 2. 依赖安装
- **lucide-vue-next**: 现代化的图标库。
- **@tailwindcss/typography**: 优化 Markdown 文章的排版样式。

## 3. 组件与页面重构
### 全局布局 (`src/App.vue`)
- 设计一个顶部导航栏，采用磨砂玻璃效果 (Backdrop blur)。
- 底部页脚保持简洁。
- 添加全局过渡动画 (`<Transition>`)。

### 首页 (`src/views/Home.vue`)
- **Hero 区域**: 简洁的自我介绍或博客标语，大号衬线字体。
- **文章列表**: 摒弃传统的列表，采用 **"卡片 + 悬停浮起"** 设计。
  - 显示文章元数据（发布时间、字数、阅读时长）。
  - 鼠标悬停时，卡片有轻微的上浮和阴影加深。

### 文章详情页 (`src/views/Post.vue`)
- **沉浸式阅读**: 隐藏不必要的干扰元素，最大化阅读区域。
- **排版优化**: 使用 `prose` (Tailwind Typography) 类，定制链接颜色、引用样式、代码块背景。
- **进度条**: 顶部添加阅读进度条。

### 管理后台 (`src/views/Admin.vue`)
- **编辑器升级**: 左侧输入 (Monospace字体)，右侧实时预览。
- **同步状态**: 在保存按钮旁添加清晰的状态指示（保存中、已同步、失败）。
- **文件列表**: 优化为表格视图，添加操作图标。

## 4. GitHub 同步功能确认
- 现有的 `putFile` 逻辑已经支持创建和更新。
- 重点在于 **编辑器体验**：输入时自动保存到本地缓存（防丢失），点击发布时才同步到 GitHub。
- 添加 **Frontmatter 支持**：允许用户在文章头部定义标题、日期、标签等元数据（需要简单的解析逻辑）。

## 5. 执行步骤
1.  安装 `lucide-vue-next` 和 `@tailwindcss/typography`。
2.  配置 `tailwind.config.js`，引入 Typography 插件和自定义字体/颜色。
3.  重构 `App.vue` 布局和导航。
4.  重构 `Home.vue` 首页视觉。
5.  重构 `Post.vue` 文章页视觉。
6.  重构 `Admin.vue`，实现分栏编辑和元数据解析。
7.  验证视觉效果和 GitHub 同步流程。
