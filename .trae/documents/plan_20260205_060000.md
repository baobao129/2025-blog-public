# 视觉优化与全面汉化计划

用户反馈页面依然"难看"（可能是 Tailwind 样式未正确加载或设计细节不够），且要求全中文。我们将进行彻底的视觉重构和汉化。

## 1. 样式加载修复
目前的截图显示样式似乎未生效（蓝色的默认链接、无布局）。这通常是因为 Tailwind 未正确扫描到文件或 CSS 未正确引入。
- **检查 `tailwind.config.js`**: 确保 `content` 路径覆盖所有 Vue 文件。
- **检查 `style.css`**: 确保 `@tailwind` 指令正确。
- **检查 `main.js`**: 确保 `style.css` 已导入。

## 2. 视觉风格重定义： "东方禅意 (Zen Minimalist)"
- **字体**: 使用思源宋体 (Noto Serif SC) 作为标题，思源黑体 (Noto Sans SC) 作为正文。
- **配色**: 
  - 背景：米宣纸色 (`#fdfbf7`)
  - 文字：墨色 (`#2c2c2c`)
  - 点缀：朱砂红 (`#c0392b`) 或 黛蓝
- **布局**: 增加更多的留白，使用卡片阴影提升层次感，圆角更加柔和。

## 3. 全面汉化
所有硬编码的英文文本都将替换为中文。

### 涉及文件与修改点：

#### `src/App.vue`
- **导航**: "Blog" -> "我的博客", "Home" -> "首页", "Admin" -> "管理".
- **页脚**: 版权信息改为中文，移除 Privacy Policy 等英文链接，改为 "关于我" 等。

#### `src/views/Home.vue`
- **Hero**: "Thoughts & Ideas" -> "思绪与随笔" (或用户自定义的中文标题).
- **副标题**: "Exploring..." -> "记录技术、设计与生活中的点滴灵感。"
- **文章卡片**: "Article" -> "文章", "Read more" -> "阅读全文", "Today" -> 日期格式化为 "YYYY年MM月DD日".

#### `src/views/Post.vue`
- **元数据**: "Author" -> "作者", "min read" -> "分钟阅读".
- **导航**: "Back to Home" -> "返回首页".
- **加载/错误**: "Loading story..." -> "正在加载文章...", "Unable to load..." -> "无法加载文章".

#### `src/views/Admin.vue`
- **标题**: "Content Management" -> "内容管理".
- **状态**: "Connected to GitHub" -> "已连接 GitHub".
- **未认证**: "Authentication Required" -> "需要身份验证", "Please upload..." -> "请上传您的 GitHub App 私钥 (.pem) 以访问管理系统。"
- **编辑器**: "New Post" -> "新建文章", "Publish" -> "发布", "Saving..." -> "保存中...".
- **表格**: "Title" -> "标题", "Size" -> "大小", "Actions" -> "操作".

## 4. 执行步骤
1.  **修复样式**：优先确保 Tailwind 正常工作（这是页面"难看"的根本原因）。
2.  **汉化 App.vue**：修改全局布局的中文。
3.  **汉化 Home.vue**：修改首页的中文和日期格式。
4.  **汉化 Post.vue**：修改文章页的中文。
5.  **汉化 Admin.vue**：修改管理后台的中文。
6.  **验证**：重启开发服务器，确保样式生效且无英文残留。
