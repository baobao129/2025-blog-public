# 开启 Monaco Editor 智能提示与升级代码演练场

我将为您开启 Monaco Editor 的智能提示 (IntelliSense) 功能，并升级代码演练场以支持 Vue 和 React 项目。

## 1. 开启智能提示 (IntelliSense)

### 原因分析
目前的编辑器缺少智能提示，是因为 Monaco Editor 依赖 Web Workers 来处理语言服务（如语法检查、自动补全），但当前项目未配置这些 Workers。

### 实施计划
- **配置 Web Workers**: 在 `src/main.js` 中手动配置 `MonacoEnvironment`。
- **引入 Worker 脚本**: 使用 Vite 的 `?worker` 语法导入 Monaco Editor 自带的 Workers：
  - `editor.worker`: 基础编辑器功能
  - `json.worker`: JSON 支持
  - `css.worker`: CSS/SCSS/LESS 支持
  - `html.worker`: HTML 支持
  - `ts.worker`: TypeScript/JavaScript 支持

## 2. 升级代码演练场 (CodePlayground.vue)

### 新增功能
- **框架切换**: 添加 "Vue" 和 "React" 选项。
- **Vue 支持**: 自动注入 Vue 3 CDN，支持 Options API 和 Composition API 写法。
- **React 支持**: 自动注入 React、ReactDOM 和 Babel (Standalone) CDN，支持直接编写 JSX。

### 修改细节
- **UI 更新**: 在顶部工具栏增加 "环境选择" 下拉菜单 (Vanilla / Vue / React)。
- **模板预设**: 切换环境时，自动填充该框架的 "Hello World" 示例代码，方便上手。
- **运行逻辑**:
  - **Vue**: 注入 `vue.global.js`，用户代码将自动挂载到 `#app`。
  - **React**: 注入 `react.development.js`, `react-dom.development.js`, `babel.min.js`，并将用户的 JS 代码作为 `<script type="text/babel">` 执行。

## 3. 验证
- 确认编辑器中输入 JS/HTML 代码时出现自动补全和错误提示。
- 确认演练场可以成功运行 Vue 的 `createApp` 和 React 的 `ReactDOM.createRoot` 示例。
