# "Cyber-Brutalist" 风格重构方案

针对您提出的彻底重构和高度创新的要求，我为您量身定制了一套 **"Cyber-Brutalist"（赛博粗野主义）** 风格的设计方案。

这套方案将彻底摒弃现有的极简/杂志风格，转而采用一种大胆、前卫、充满科技感和原始力量的视觉语言。它强调高对比度、单色调与霓虹色的碰撞、非对称网格布局、以及大量的故障艺术（Glitch Art）和微交互动效。

## 1. 视觉与设计理念 (Design Concept)

*   **核心风格**: **Cyber-Brutalist (赛博粗野主义)**
*   **设计隐喻**: "数字废墟中的全息终端"。想象一个复古未来的数据终端，界面充满原始的代码感、裸露的网格线和高饱和度的信号色。
*   **配色方案 (Color Palette)**:
    *   **背景**: 深邃黑 (`#050505`) 或 终端绿黑 (`#0a0f0a`)。
    *   **主色**: 霓虹酸性绿 (`#CCFF00`) 或 赛博洋红 (`#FF00FF`)，用于强调和交互。
    *   **辅助色**: 水泥灰 (`#888888`)，用于网格线和次要信息。
    *   **文字**: 纯白 (`#FFFFFF`) 或 终端绿 (`#00FF00`)。
*   **字体系统 (Typography)**:
    *   **标题**: `Space Mono` 或 `Press Start 2P`（像素风格），强调机械感和复古感。
    *   **正文**: `JetBrains Mono` 或 `IBM Plex Mono`（等宽字体），营造代码编辑器的氛围。
*   **图形元素**:
    *   **网格线 (Grid Lines)**: 显眼的 1px 边框，将页面分割成不规则的区块。
    *   **故障效果 (Glitch)**: 鼠标悬停时的图像错位、文字闪烁。
    *   **Marquee (跑马灯)**: 滚动的文字带，用于展示标签或公告。

## 2. 交互与布局重构 (Layout & Interaction)

### 首页 (Home.vue)
*   **布局**: 摒弃传统的垂直列表，采用 **"Bento Grid" (便当盒网格)** 布局。文章卡片大小不一，像拼图一样错落排列。
*   **Hero 区域**:
    *   巨大的、占据半屏的加粗标题，带有打字机效果。
    *   背景是一个基于 WebGL 或 Canvas 的交互式粒子场，随鼠标移动产生扰动。
*   **文章卡片**:
    *   **默认态**: 黑白线框模式，仅显示标题和元数据。
    *   **Hover态**: 瞬间反色（黑底变白底，或变霓虹色），图片以故障效果浮现，光标变为特殊的十字准星。

### 文章详情页 (Post.vue)
*   **沉浸式阅读**:
    *   **左侧**: 固定的元数据栏（日期、标签、阅读进度），像代码编辑器的行号区域。
    *   **中间**: 正文区域，采用高对比度的等宽字体排版。
    *   **右侧**: 动态生成的目录树，带有连接线效果。
*   **代码块**: 采用极简的终端风格，黑色背景，霓虹色高亮。
*   **进度指示器**: 页面边缘的像素化进度条，随滚动填充。

### 导航与全局组件
*   **导航栏**: 浮动的玻璃态控制台，按钮采用物理开关的拟物设计或像素化图标。
*   **光标**: 自定义 SVG 光标（如十字准星、像素手型），点击时有波纹扩散效果。

## 3. 技术实现路径 (Roadmap)

1.  **基础设施升级**:
    *   引入 `Google Fonts` (Space Mono, JetBrains Mono)。
    *   配置 Tailwind 的自定义主题（颜色、字体、边框）。
    *   引入 `anime.js` 或 `gsap` 用于复杂的动效控制。

2.  **组件重写**:
    *   **Global Styles**: 重置所有默认样式，应用黑色背景和等宽字体。
    *   **App.vue**: 实现全局的网格背景和自定义光标。
    *   **Home.vue**: 实现 Bento Grid 布局和故障风文章卡片。
    *   **Post.vue**: 重构为三栏式 IDE 风格布局。

3.  **创新动效开发**:
    *   **GlitchImage.vue**: 封装一个故障效果图片组件。
    *   **TypewriterText.vue**: 封装打字机效果文字组件。
    *   **Marquee.vue**: 封装无限滚动跑马灯组件。
