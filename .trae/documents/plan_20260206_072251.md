# 代码演练场页面重构

我将把代码演练场从 `Post.vue` 中的全屏模式剥离出来，使其成为一个独立的页面路由。这样可以提供更好的全屏体验，并且与文章详情页解耦。

## 1. 创建演练场视图 (Playground.vue)
- 创建 `src/views/Playground.vue`。
- 将 `src/components/CodePlayground.vue` 中的大部分布局逻辑迁移到这个新视图中。
- **调整布局**：去除顶部的博客导航栏（使其成为真正的全屏应用），让编辑器和预览区占据 100% 的视口空间。
- **获取文章内容**：通过路由参数（`:filename`）获取文章内容，并在组件挂载时加载，以便在左侧显示文章详情。

## 2. 更新路由配置 (router/index.js)
- 添加新路由 `/playground/:filename`。
- 指向新的 `Playground.vue` 组件。
- 设置 `meta: { layout: 'fullscreen' }` 以确保隐藏全局布局元素（如页眉/页脚）。

## 3. 修改文章详情页 (Post.vue)
- 移除原来的 `CodePlayground` 组件引用和全屏模态框逻辑。
- 修改“演练模式”按钮的点击事件，使其跳转到 `/playground/${filename}`。

## 4. 验证
- 确认点击文章页面的“演练模式”会跳转到新页面。
- 确认新页面中能正常加载文章内容、编辑器和预览区。
- 确认新页面的布局不再受博客主站样式的限制，实现真正的全屏沉浸式体验。
