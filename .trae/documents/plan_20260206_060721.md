# 添加代码演练场并恢复编辑器默认高亮

我将为您实现一个在线代码演练场（Playground），并还原编辑器的默认高亮设置。

## 1. 还原 Monaco Editor 默认高亮 (MonacoEditor.vue)
- **修改主题**：移除之前自定义的 `one-dark` 主题配置。
- **恢复默认**：将编辑器主题设置回 Monaco Editor 自带的 `vs-dark`（VS Code 深色主题）。
- **增加灵活性**：添加 `language` 属性支持，以便代码演练场可以切换 HTML/CSS/JS 语言高亮。

## 2. 创建代码演练场组件 (CodePlayground.vue)
创建一个新的组件 `src/components/CodePlayground.vue`，功能包括：
- **多语言编辑**：提供 HTML、CSS、JavaScript 三个标签页，使用 `MonacoEditor` 进行编辑。
- **实时预览**：右侧使用 `<iframe>` 实时渲染代码运行结果。
- **沙箱环境**：代码仅在本地浏览器内存中运行，不与后端交互，确保安全且不上传 Git。
- **下载功能**：提供“下载源码”按钮，将当前代码打包为一个 `.html` 文件供用户下载。

## 3. 集成到文章详情页 (Post.vue)
- **入口按钮**：在文章页面右下角添加一个悬浮按钮（类似于“代码实验室”图标）。
- **模态框展示**：点击按钮后，以全屏或大弹窗形式打开代码演练场，方便用户在阅读文章的同时进行代码实验。

## 4. 验证
- 确认编辑器页面 (`Editor.vue`) 的代码高亮恢复为默认的 VS Code 风格。
- 确认文章页面 (`Post.vue`) 可以打开演练场，并且 HTML/CSS/JS 代码能正确运行和下载。
