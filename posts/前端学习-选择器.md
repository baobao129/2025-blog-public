# 选择器

选择器：帮助你精准的选中想要的元素

## 简单选择器

**1. ID选择器**
``` CSS
/* 为所有id为intro的元素设置字体加粗 */
#intro {
  font-weight:bold
}

<p id = "intro">This is a paragraph of introduction.</p>
```
**2. 元素选择器**
```CSS
/* 为所有段落文字设置蓝色 */
p {
color: blue;
}

<p>This is a paragraph of introduction.</p>
```
**3. 类选择器**
```CSS
/* 为所有段落文字设置红色 */
red {
color: red;
}

<p class = "red">This is a paragraph of introduction.</p>
```

**4. 通配符选择器**

*，选中所有的元素

```CSS
/* 为所有元素设置红色 */
* {
color: red;
}

<p>This is a paragraph of introduction.</p>
```
**5. 属性选择器**

根据属性名和

```CSS
/* 选中所有具有href属性的元素 */
[href]{
    color:red;

}
<a href = "https://www.sina.com">新浪</a>
<a href = "https://www.baidu.com">百度</a>
```
全部变红

```CSS
/* 选中具有href属性的元素，并且href的值为"https://www.sina.com" */
[href = "https://www.sina.com"]{
    color:red;
}
<a href = "https://www.sina.com">新浪</a>
<a href = "https://www.baidu.com">百度</a>
```
只有新浪变红

```CSS
[class ~= "b"]{
  color: red;
  font-weight: bold;
  display: block;
}

  <a class = "a b c d" href="https://www.baidu.com">百度</a>
  <a class = "a c d" href="https://www.jd.com">京东</a>
```
只会百度变红。

**[attr~=value]**
表示带有以attr命名的属性元素，并且该属性是一个以空格作为分隔的值列表，其中至少一个值匹配"value"。

>更多：https://developer.mozilla.org/zh-CN/docs/Web/CSS/Reference/Selectors/Attribute_selectors

**6. 伪类选择器**

选中某些元素的某种状态

  1. link：超链接未访问时的状态
  2. visited：超链接访问过后的状态

  ```CSS
  a:link {
    color: chocolate;
  }

  a:visited {
    color: red;
  }


    <a href="https://www.baidu.com">百度</a>
    <a href="https://www.jd.com">京东</a>

  ```

  3. hover：鼠标悬停状态

  ```CSS
  /* 选中鼠标悬停时的a元素 */
  a:hover{
    color:red;
  }

    <a href="https://www.baidu.com">百度</a>
    <a href="https://www.jd.com">京东</a>
  ```

  4. active： 激活状态，鼠标按下状态

  ```CSS
  /* 鼠标按下时的a元素 */
  a:hover{
    color:red;
  }

    <a href="https://www.baidu.com">百度</a>
    <a href="https://www.jd.com">京东</a>

  ```

<mark>**如果需要设置属性，需要按照这样的顺序进行书写**</mark>

爱恨法则：love hate


**7. 伪元素选择器**

生成并选中某个元素内部的第一个子元素或最后一个子元素

before：在元素中生成一个子元素，并且子元素一定是第一个元素。

after：在元素中生成一个子元素，并且子元素一定是最后一个元素。

```CSS

span::before {
  /* content，只有before和after中才能够使用，表示生成的子元素中的内容 */
  content: "《";
  color: red
}

span::after {
  contetn: "》";
}

  <p>
      国务院办公厅转发民政部等10部门<span>关于进一步加强养老机构安全管理的意见</span>。
  </p>
```

## 选择器的组合

1. 并且

    连续一起写，如`p.red { color:red }`，两个选择器连续写在一起。

2. 后代元素 —— 空格

```CSS

div.red li {
    color: #008c8c;
}

<div class = "red">
    Lorem.
    <ul>
        <li>Lorem.</li>
        <li>Harum.</li>
        <li>Eum!</li>
        <li>Impedit?</li>
    </ul>
</div>
```

3. 子元素 —— >

```CSS
/* div元素下的ul元素下的li元素 */
div>ul>li{
  color: #008c8c;
}
```

4. 相邻兄弟元素 —— +

```html
<style>
  .special {
    color: red;
  }

  /* 类选择器special的下一个li元素颜色为#008c8c */
  .special+li {
    color: #008c8c;
  }
<style>

<div>
  <ul>
    <li>Lorem!</li>
    <li class = "special">nostrum porro?</li>
    <li>dolore!</li>
    <li>rem!</li>
    <li>quo?</li>
  </ul>
</div>
```

5. 兄弟元素 —— ~

```html
<style>
  .special {
    color: red;
  }

  /* 类选择器special的后面所有li兄弟元素颜色为#008c8c */
  .special~li {
    color: #008c8c;
  }
<style>

<div>
  <ul>
    <li>Lorem!</li>
    <li class = "special">nostrum porro?</li>
    <li>dolore!</li>
    <li>rem!</li>
    <li>quo?</li>
  </ul>
</div>
```
