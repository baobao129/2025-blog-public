# 选择器

选择器：帮助你精准的选中想要的元素

## 简单选择器

1. ID选择器
``` CSS
/* 为所有id为intro的元素设置字体加粗 */
#intro {
  font-weight:bold
}

<p id = "intro">This is a paragraph of introduction.</p>
```
2. 元素选择器
```CSS
/* 为所有段落文字设置蓝色 */
p {
color: blue;
}

<p>This is a paragraph of introduction.</p>
```
3. 类选择器
```CSS
/* 为所有段落文字设置红色 */
red {
color: red;
}

<p class = "red">This is a paragraph of introduction.</p>
```

4. 通配符选择器

*，选中所有的元素

```CSS
/* 为所有元素设置红色 */
* {
color: red;
}

<p>This is a paragraph of introduction.</p>
```
5. 属性选择器

根据属性名和

```CSS
/* 选中所有具有href属性的元素 */
[href]{
    color:red;

}
<a href = "https://www.sina.com">新浪</a>
<a href = "https://www.baidu.com">百度</a>
```
全部变红

```CSS
/* 选中具有href属性的元素，并且href的值为"https://www.sina.com" */
[href = "https://www.sina.com"]{
    color:red;
}
<a href = "https://www.sina.com">新浪</a>
<a href = "https://www.baidu.com">百度</a>
```
只有新浪变红

```CSS
[class ~= "b"]{
  color: red;
  font-weight: bold;
  display: block;
}

  <a class = "a b c d" href="https://www.baidu.com">百度</a>
  <a class = "a c d" href="https://www.jd.com">京东</a>
```
只会百度变红。

**[attr~=value]**
表示带有以attr命名的属性元素，并且该属性是一个以空格作为分隔的值列表，其中至少一个值匹配"value"。

>更多：https://developer.mozilla.org/zh-CN/docs/Web/CSS/Reference/Selectors/Attribute_selectors

## 选择器的组合

1. 后代元素 —— 空格
2. 子元素 —— >
3. 相邻兄弟元素 —— +
4. 兄弟元素 —— ~
