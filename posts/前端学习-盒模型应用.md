# 盒模型应用

## 改变宽高范围

默认情况下，width 和 height 设置的时内容盒的宽高。

>页面重构：将psd文件（设计稿）制作为静态页面

衡量设计稿尺寸时，通常使用的是边框盒的尺寸，而设置width 和 height时，则是设置的内容盒的尺寸。

### 解决方法
- 精确计算
- 使用CSS3：box-sizing 用于改变宽高的影响范围
| 参数 | 作用 |
| --- | --- |
| content-box | 默认值，宽高值设置范围是内容盒 |
| border-box | 宽高值设置范围是边框盒，是整体div盒 |

```CSS
div {
      width: 236px;
      height: 51px;
      color: #a7a7a7;
      background: #2d2e36;
      /* 改变宽高的影响范围 */
      box-sizing: border-box;
}
```

## 改变背景覆盖范围

默认情况下，背景覆盖边框盒，及整体的div内容

可以通过**background-clip**的参数进行修改
| 参数 | 范围|
| --- | --- |
| border-box | 默认值，设置范围是边框盒 |
| padding-box | 设置范围是填充盒 |
| content-box | 设置范围是内容盒 |

## 溢出处理

**overflow**：控制内容溢出后的处理方式，参数设置为**scroll**，会显示滚动条。可以单独设置x轴或者是y轴的滚动条显示。

```CSS
div {
      overflow-x: hidden;
      overflow-y: scroll;
}
```
即设置为x轴隐藏，y轴显示。
```CSS
div {
      overflow-y: auto
}
```
参数设置为**auto**时，将自动根据文字的溢出情况生成滚动条，溢出时生成，不溢出时自动隐藏。

## 断词规则

**word-break**,会影响文字在什么位置被截断换行

| 参数 | 截断位置|
| --- | --- |
| normal | 默认值，中文、韩文、日文在文字位置截断，其他语言在单词位置截断 |
| break-all | 截断所有，所有的字符都在文字处截断|
| keep-all | 保持所有，所有文字都在单词之间截断，中文直到空格才会截断 |

```CSS
div {
      word-break: normal/ break-all/ keep-all
}
```

## 空白处理

>例子：热点新闻中，一行的最后一般都有...来进行结尾，如何进行实现

- **white-space: nowrap** 设置一行的结尾不进行换行
- **overflow: hidden** 溢出隐藏
- **text-overflow: ellipsis** 文本溢出设置为点

*只能够设置单行*

```CSS
li {
      border-bottom: 1px dashed #ccc;
      line-height: 2;
      border-left: 3px solid #008c8c;
      padding-left: 10px;
      margin: 1em 0;
      width: 200px;
      white-space: nowarp;
      overflow: hidden;
      text-overflow: ellipsis;
}
```

white-space: pre 不会进行空白折叠，比如在div中换行

```html
<div>
      var i = 1;
      alert("a");
</div>
```
默认显示为

```JavaScript
var i = 1;alert("a");
```

当设置white-space: pre时，会显示为
```JavaScript
var i = 1;
alert("a");
```
