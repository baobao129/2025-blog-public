# 盒模型应用

## 改变宽高范围

默认情况下，width 和 height 设置的时内容盒的宽高。

>页面重构：将psd文件（设计稿）制作为静态页面

衡量设计稿尺寸时，通常使用的是边框盒的尺寸，而设置width 和 height时，则是设置的内容盒的尺寸。

### 解决方法
- 精确计算
- 使用CSS3：box-sizing 用于改变宽高的影响范围
| 参数 | 作用 |
| --- | --- |
| content-box | 默认值，宽高值设置范围是内容盒 |
| border-box | 宽高值设置范围是边框盒，是整体div盒 |

```CSS
div {
      width: 236px;
      height: 51px;
      color: #a7a7a7;
      background: #2d2e36;
      /* 改变宽高的影响范围 */
      box-sizing: border-box;
}
```

## 改变背景覆盖范围

默认情况下，背景覆盖边框盒，及整体的div内容

可以通过**background-clip**的参数进行修改
| 参数 | 范围|
| --- | --- |
| border-box | 默认值，设置范围是边框盒 |
| padding-box | 设置范围是填充盒 |
| content-box | 设置范围是内容盒 |

## 溢出处理

**overflow**：控制内容溢出后的处理方式，参数设置为**scroll**，会显示滚动条。可以单独设置x轴或者是y轴的滚动条显示。
```CSS
{
 overflow-x:hidden;
 overflow-y:scroll;
}
```
即设置为x轴隐藏，y轴显示。
```CSS
{
 overflow-y:auto
}
```
参数设置为**auto**时，将自动根据文字的溢出情况生成滚动条，溢出时生成，不溢出时自动隐藏。
